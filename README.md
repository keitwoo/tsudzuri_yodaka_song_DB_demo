# 綴よだか　歌唱コーパス v0.1 
![yodaka_DL6](https://user-images.githubusercontent.com/32325910/147927243-30cda266-4438-4e65-870c-876b95abb10d.png)

名古屋工業大学が作成した日本語音声データベース「NITSONG070F001」を元に  
童謡などの著作権が失効した楽曲を歌い上げた歌唱コーパスです
